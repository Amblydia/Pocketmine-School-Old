plugin.yml

First create a folder and name it whatever you want because this folder will be your plugin.

In the folder create a file and name it plugin.yml

In this file you will add:

  1. name: Name of Your Plugin
  2. author: Your Name
  3. api: 3.0.0
  4. version: 1.0  #This is where you put the version of the plugin
  5. main: ExampleName/Main
  6. # "ExampleName" will be a folder in the src folder and "Main" will be the file in the "ExampleName" folder
  7.  
  8. # Commands If ur plugin has commands do this:
  9. command:
  10. 	test:  #"test" this is the name of the command
  11. 		description: Put any description
  12. 		usage: /test
  13. 	test2:  #Second command
  14. 		description: Put any Description
  15. 		usage: /test2
  16.  
  17. # Permissions 
  18. permissions:
  19. 	test.command:
  20. 		default: true
  21. 	test2.command:
  22. 		default: true

So after you add your plugin.yml make a folder and name it src, in this folder you will add a new folder called "ExampleName" (were nameing it "ExampleName" because that is what we named it in the plugin.yml).

Now in the "ExampleName" folder we add a new file and name it "Main.php" (were nameing it "Main.php" because that is what we named it in the plugin.yml and the .php is the extension).

Now your file structure should look like this:

        
          PluginName 
              ├── plugin.yml 
              └── src
                   └── ExampleName
                            └── Main.php
