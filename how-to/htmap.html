Introduction

What's a Pocketmine Plugin?

A plugin is an external module which can be “plugged in” to your server to add custom features, change default behaviours and more.


Before You Start

  • You should know the PHP language.
  • You should have a working Pocketmine server.
  • You should have the DevTool plugin.

    • Ok lets start.

    plugin.yml

    First create a folder and name it whatever you want because this folder will be your plugin.

    In the folder create a file and name it plugin.yml

    In this file you will add:

    1. name: Name of Your Plugin
    2. author: Your Name
    3. api: 3.0.0
    4. version: 1.0 #This is where you put the version of the plugin
    5. main: ExampleName/Main
    6. # For "main" statement above, find the main class path, but exclude "src"
    7.  
    8. # Commands: If ur plugin has commands: Do this:
    9. command:
    10. 	cmdname: #"cmdname" this is the name of the command
    11. 		description: Put any description
    12. 		usage: /cmdname
    13. 	cmd2name: #Second command
    14. 		description: Put any Description
    15. 		usage: /cmd2name
    16.  
    17. # Permissions 
    18. permissions:
    19. 	cmdname.command:
    20. 		default: true
    21. 	cmd2name.command:
    22. 		default: true
